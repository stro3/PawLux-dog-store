# PawLux — Shopify Development Store

Premium dog accessories and wellness brand built on Shopify (Dawn theme).

## Project Structure

| Folder | Contents |
|---|---|
| `brand/` | Brand guide, logo files |
| `products/` | Product CSV, descriptions, AI-generated product images |
| `content/` | Homepage sections, policy pages, Track Order (HTML) |
| `theme/` | CSS overrides, Liquid snippets, email popup |
| `seo/` | Meta titles and descriptions for all pages |
| `guides/` | Step-by-step setup, app config, Sidekick AI, QA checklists |

## Quick Start

1. Read `guides/01-store-setup.md` to create your dev store
2. Upload `brand/logo/` files to Shopify
3. Upload product images from `products/images/` to each product
4. Import `products/products.csv` via Shopify Admin > Products > Import
5. Paste content from `content/` into Shopify pages
6. Add code from `theme/` to your Dawn theme customization
7. Follow `guides/02-app-setup.md` to install apps
8. Use `guides/06-sidekick-ai.md` for AI-powered optimization
9. Run through `guides/05-qa-checklist.md` before launch
