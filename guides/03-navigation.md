| Menu Item | Link Type | Link Target |

### How to Add Dropdown Items
1. Click "Add menu item" and create "Beds & Furniture"
2. Drag it slightly to the right under "Collections" to make it a sub-item
3. Repeat for all sub-collection items
4. Save the menu

## Breadcrumb Navigation

Dawn theme includes breadcrumbs by default. Verify they appear on:
- Product pages (Home > Collection > Product)
- Collection pages (Home > Collection)
- Content pages (Home > Page)

---

## Mobile Navigation

Dawn generates mobile navigation automatically from the main menu. Verify:
- Hamburger menu icon appears on screens under 990px
- Dropdown items expand correctly on tap
- All links are reachable within 2 taps

## Common Mistakes

- Creating too many top-level menu items (keep it under 7 for readability)
- Not testing dropdowns on mobile after making changes
- Forgetting to add policy pages to the footer menu
- Linking to pages that have not been created yet (results in 404 errors)
